Hero App Store
Description
Hero App Store is a modern React-based web application that allows users to explore popular apps, install them, and manage their installed ones with ease.
The project is responsive, dynamic, and interactive, featuring essential functionalities such as Local Storage integration, sorting options, chart visualizations, and toast notifications for a smooth user experience.
Technologies Used
•	HTML5 → Provides the basic structure of the web pages
•	CSS3 → Adds styling and visual appeal
•	Tailwind CSS → Used for a responsive, utility-first design
•	DaisyUI → Pre-built Tailwind components for faster UI development
•	JavaScript (ES6+) → Handles core logic and interactivity
•	React.js → Component-based frontend framework for building the app
•	React Router → Enables seamless client-side navigation
•	Axios → For data fetching and API requests
•	React Toastify → Displays interactive toast notifications
•	Recharts → Used for clean and engaging data visualizations
Developed by Arafat, using React and Tailwind CSS as part of a student project.
